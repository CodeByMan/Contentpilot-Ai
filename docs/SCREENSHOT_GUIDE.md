# Screenshot Guide for ContentPilot

Capture these screens before publishing the GitHub repository or portfolio page.

## Must-have screenshots

1. **Landing Page** — show the main SaaS promise and CTA.
2. **Login Page** — show polished auth experience.
3. **Admin Dashboard** — show metrics, popular templates, plans, recent documents.
4. **Client Dashboard** — show word usage, plan badge, templates, recent drafts.
5. **Template Builder** — show dynamic AI template configuration.
6. **AI Content Generator** — show language, model, length, fields, and generated output.
7. **Generated Documents** — show saved output history.
8. **Image Generator** — show prompt-to-image workflow.
9. **Audio Generator** — show text-to-speech workflow.
10. **Invoice PDF** — show billing/business functionality.
11. **Mobile View** — show responsive dashboard/sidebar.
